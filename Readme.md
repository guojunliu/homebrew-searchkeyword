## 介绍

Mac下扫描文件夹中所有文件是否包含关键字的工具

## 安装

需要提前安装[Homebrew](https://brew.sh/)，然后在终端中运行以下代码

`brew install guojunliu/searchkeyword/searchkeyword`


## 使用

```
searchkeyword [keyword]

示例：
searchkeyword UIWebView
```
